# HangzStore Bot – All-in-One Telegram Bot

Bot Telegram multifungsi dengan sistem **Coin & Script**, **Activator VIP AnimeLovers**, **Activator Premium Alight Motion**, serta **Obfuscate JavaScript** dengan proteksi tinggi. Bot ini dirancang untuk komunitas script dan developer, dilengkapi dengan force join, admin chat, owner dashboard, dan fitur lengkap lainnya.

---

## ✨ Fitur Utama

- ✅ **Sistem Coin & Script** – Tukar koin dengan script premium, beli koin, klaim harian, referral, dan leaderboard.
- ✅ **AnimeLovers VIP Activator** – Aktivasi VIP AnimeLovers untuk durasi 1 bulan, 3 bulan, atau 1 tahun (dengan batasan per user).
- ✅ **Alight Motion Premium Activator** – Aktivasi lisensi Alight Motion via API Key (khusus VIP).
- ✅ **Obfuscate JavaScript** – Proteksi kode JS dengan `javascript-obfuscator` (proteksi ekstrem, anti‑debug, self‑defending, dan support kadaluwarsa).
- ✅ **Force Join** – Wajib join channel/group sebelum menggunakan bot.
- ✅ **Admin Chat** – Komunikasi dua arah antara user dan admin.
- ✅ **Owner Dashboard** – Kelola user, script, redeem code, backup database, dan lain-lain.
- ✅ **Auto‑Install Dependency** – Secara otomatis menginstal modul yang kurang saat pertama kali dijalankan.

---

## 📋 Persyaratan

- **Node.js** v16 atau lebih baru
- **npm** atau **yarn**
- Akun Telegram dan **Bot Token** (dari [@BotFather](https://t.me/BotFather))
- (Opsional) Akun **Railway** untuk deployment online
